Preguntas Frecuentes

¿Qué es Lira?

Lira es una aplicación móvil diseñada para facilitar la comunicación bidireccional entre personas sordas y oyentes, traduciendo en tiempo real el Lenguaje de Señas Salvadoreña (LESSA) al español y viceversa, utilizando inteligencia artificial.

¿Cómo funciona la traducción en tiempo real?

La aplicación utiliza la cámara de tu dispositivo para captar los movimientos del lenguaje de señas y el micrófono para la voz. Nuestra IA procesa esta información y la traduce al idioma deseado, mostrándola en pantalla o a través de un avatar interactivo.

¿Lira es compatible con otros lenguajes de señas?

Actualmente, Lira está optimizada para el Lenguaje de Señas Salvadoreña (LESSA). Estamos trabajando en la incorporación de otros lenguajes de señas en futuras actualizaciones.

¿Necesito una conexión a internet para usar Lira?

Para la funcionalidad completa de traducción en tiempo real y acceso a todas las características, se recomienda una conexión a internet activa. Algunas funciones básicas podrían estar disponibles sin conexión.
